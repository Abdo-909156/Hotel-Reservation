# Hotel Reservation System

A console-based hotel reservation management system built in C.

## About

A terminal application that allows users to make hotel reservations and automatically calculates the total cost based on the selected room type and meal plan. All reservations are stored and can be displayed at any time.

## Features

- Make a new hotel reservation
- Choose meal plan — Full Board, Breakfast only, or No Meals
- Automatic cost calculation based on room type and meal plan
- View all stored reservations
- Console-based interface with simple input/output

## Tech Stack

- **Language:** C
- **Interface:** Console / Terminal

## How to Run

### Requirements
- A C compiler such as GCC or any IDE that supports C (e.g. Code::Blocks, Dev-C++)

### Steps
1. Clone the repository
   ```
   git clone https://github.com/Abdo-909156/Hotel-Reservation.git
   ```
2. Compile the source file
   ```
   gcc hotel.c -o hotel
   ```
3. Run the program
   ```
   ./hotel
   ```
